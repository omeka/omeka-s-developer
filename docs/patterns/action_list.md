# Action List

An action list is a series of interactive icons representing actions that can be performed on an associated subject. Examples appear on the resource browse views within each table row.

![A row from an item browse view, "Interview with Joseph Tyler Gayheart, May 28, 2010" opposite a set of icons: pencil for edit, trash can for delete, and an ellipsis for showing details.](../img/action_links.jpg)

</ul>
